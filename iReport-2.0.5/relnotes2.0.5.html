iReport Release 2.0.5
March 17, 2007

The following changes have been made to software since release 2.0.4
  • Updated to JasperReports 2.0.5
  • New element border editor supporting the new JasperReports pen definitions
  • Support for the new SWF (Flash) exporter (experimental, FLEX 3 SDK required)
  • Included JasperServer plugin 2.1.0
  • Several bug fixes
Contributors this time:
  • Giulio Toffoli
Thanks for your suggestions and your on going support using iReport!

Giulio
